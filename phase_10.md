## Current Architecture (for any agent reading this)

### Key Functions in `generate_dashboard.py`

> ⚠️ This is the ACTUAL current code structure. Older phase docs may reference functions like `build_context()` or CSS classes like `glass-card` — those no longer exist.

| Function | Line | Purpose |
|:--|:--|:--|
| `_latest_btc_alert(alerts)` | ~401 | Returns the most recent BTC alert from the alerts list |
| `build_verdict_context(alerts, portfolio)` | ~408 | **Main verdict builder.** Extracts direction, entry/tp1/stop, active reason codes, risk gate checks (5 conditions), and confluence probes (10 items). Returns a dict used by `generate_html()` |
| `generate_svg_equity(curve)` | ~371 | Renders an SVG polyline sparkline from the portfolio equity curve |
| `generate_html()` | ~441 | **Main renderer.** Calls all the above, builds the entire HTML page, writes to `dashboard.html` |

### How the Verdict Card is Assembled (in `generate_html()`)

The Verdict card sections are built in this order inside `verdict_html`:

```
1. Direction badge (LONG/SHORT/WAIT)
2. Live BTC Price + PnL overlay (id="livePrice", id="livePnL", id="distTP1", id="distStop", id="liveSpread")
3. Conviction Signals (reason code pills)
4. Confluence Radar (10 probe grid with progress bar) ← Phase 11 enhances this
5. Trade Safety gate (5-check list with GREEN/AMBER/RED)
6. Execute button (styled RED if gate is RED)
7. Execute confirmation modal (3-second countdown)
```

### WebSocket Handler (`connectWS()`)

The `ws.onmessage` handler receives this payload every ~2 seconds:
```json
{
    "orderbook": {"bid": 64300, "ask": 64310, "mid": 64305, "spread": 10},
    "portfolio": {"balance": 9279, "positions": [], "max_drawdown": 0.069},
    "alerts": [...],
    "stats": {"win_rate": 60, "profit_factor": 4.8, "avg_r": 1.2, "streak": -1}
}
```

It updates:
- `state.livePrice` and `state.spread` from `data.orderbook`
- Live Tape tiles (BTC Mid, Spread, Confluence, Radar, Balance, Win Rate, Profit Factor, Risk Gate)
- Calls `updateLivePrice()` which updates the Verdict card's live price, distances, and PnL
